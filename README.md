# 💼 Accounting & ERP System

## 📌 Overview
A web-based accounting and ERP system developed to manage financial operations, reporting, and business workflows efficiently.

---

## 🚀 Features
- General Ledger Management  
- Chart of Accounts  
- Voucher Entry System  
- Financial Reports (Profit/Loss, Balance Sheet)  
- User Role & Permission Management  

---

## 🛠️ Tech Stack
- PHP (Laravel / CodeIgniter)  
- MySQL / PostgreSQL  
- JavaScript  
- HTML / CSS  

---

## ⚙️ System Modules
- Accounts Management  
- Transaction Processing  
- Reporting Dashboard  
- User Access Control  

---

## 🏗️ Architecture
- MVC Architecture  
- Modular Design  
- REST-ready structure  

---

## ▶️ How to Run
1. Clone the repository  
2. Setup database  
3. Configure database connection  
4. Run on local server (XAMPP / WAMP / Laravel serve)  

---

## 📈 Purpose
To digitize financial processes, reduce manual work, and improve reporting accuracy.
